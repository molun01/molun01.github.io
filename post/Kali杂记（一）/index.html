Kali杂记（一）

2021-11-20 #系统

安装Kali后的基本配置

  1. 普通用户kali/kali登录，sudo su提权

  2. passwd，修改root用户密码

  3. 配置中文环境

    • /usr/sbin/dpkg-reconfigure locales
    • 选择en_US-utf8、zh_CN-gbk、zh_CN-utf8
    • zh_CN-utf8设为默认编码

  4. 换国内源

    • mousepad /etc/apt/sources.list
    • 注释Kali官方源，并添加以下：
    #中科大
deb http://mirrors.ustc.edu.cn/kali kali-rolling main non-free contrib
deb-src http://mirrors.ustc.edu.cn/kali kali-rolling main non-free contrib
#deb http://mirrors.ustc.edu.cn/kali-security kali/updates main contrib non-free

  5. 检查更新apt-get update && apt-get upgrade && apt-get clean

  6. 装中文字体apt-get install ttf-wqy-microhei ttf-wqy-zenhei xfonts-wqy

  7. 重启中文环境生效reboot

  8. 安装中文输入法apt-get install ibus ibus-pinyin

  9. 启动ssh并设置开机自启systemctl enable --now ssh

  10. 纯命令行启动systemctl set-default multi-user.target

    或恢复图形界面启动systemctl set-default graphical.target

    然后reboot
