研究课题：在线考试系统

一款JAVAWEB考试系统，可自由定制功能，划分用户身份权限，管理题目和试卷。可以随机抽取题目生成试卷，计时器（到时自动交卷）功能，交卷时判断试卷是否有未答题，给出警告。自动计算得分，学生可以查看试卷得分。自由练习时，查看错题以查缺补漏。
